## Project Overview
This project is a **database management system for the Indian Premier League (IPL)** developed to store, manage, and retrieve cricket-related data such as teams, players, matches, and statistics. It demonstrates practical use of **database design and backend development** concepts.

## What Was Done
- Designed a structured **relational database** for IPL data
- Created tables for teams, players, matches, and records
- Implemented **CRUD operations** using PHP
- Connected frontend with database for data handling
- Added screenshots to show system working

## What Was Achieved
- Efficient storage and retrieval of IPL data
- Hands-on experience with **DBMS concepts**
- Improved understanding of **SQL queries and relationships**
- Built a functional backend database application
